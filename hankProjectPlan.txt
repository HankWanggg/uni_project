git command:

1. git status （检查当前的git状态）
2. git add -A  (把当前本地所有的changes打包到一个package里面)
3. git commit -a -m "comment" （把package里的所有changes保存下来 并且attach上comment）
4. git push (把已经commit的内容 发送到github上面)
